# movie-recommendation-system
A Streamlit app that recommends movies based on genre similarity and displays posters in a clean grid layout.

🎞️Features:
- 🎬 Select a movie and get 5 similar recommendations
- 🍿 Sidebar with filters and controls
- 🖼️ Posters displayed neatly in rows of 3
- ⚡ Built with Python, Pandas, Scikit‑learn, and Streamlit

▶️ Usage
Run the app with:
python -m streamlit run movie_recommendation.py


Then open the local URL shown in your terminal (usually http://localhost:8501) to use the app.

🛠️ Requirements
- Python 3
- Streamlit
- Pandas
- Scikit‑learn
